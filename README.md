Cowdia is cross-platform 3D rendering engine.


## Quick Start
First of all, clone the repository.
```bash
git clone --recursive https://github.com/JYPark09/Cowdia
cd Cowdia
```

And then, build the project.
```bash
mkdir build
cd build
cmake ..
make
```
